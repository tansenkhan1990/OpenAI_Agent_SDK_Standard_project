# Scripts Directory

Utility scripts for development, testing, and deployment.

## Available Scripts

### setup.sh
Initialize the project environment
```bash
./scripts/setup.sh
```

### test.sh
Run all tests with coverage
```bash
./scripts/test.sh
```

### format.sh
Format code with black and ruff
```bash
./scripts/format.sh
```

### lint.sh
Run linting checks
```bash
./scripts/lint.sh
```

### run-local.sh
Run the application locally
```bash
./scripts/run-local.sh
```

### run-prod.sh
Run in production mode
```bash
./scripts/run-prod.sh
```

**Note:** Add execute permission to scripts:
```bash
chmod +x scripts/*.sh
```
